# RDP Configurator - Интерактивный конфигуратор RDP файлов

## Описание

Это Python приложение позволяет интерактивно создавать и редактировать файлы конфигурации Remote Desktop Protocol (RDP) с подробным объяснением каждого параметра.

## Функциональность

- ✅ **Создание новых RDP конфигураций** - интерактивный процесс с объяснением каждого параметра
- ✅ **Загрузка и редактирование** - открытие существующих RDP файлов для модификации
- ✅ **Справочная информация** - подробное описание каждого параметра по запросу
- ✅ **Сохранение конфигураций** - экспорт в стандартный формат RDP файлов
- ✅ **Интуитивный интерфейс** - простое и понятное меню

## Параметры, которые поддерживаются

### Основные параметры подключения
- `full address` - IP адрес или имя компьютера
- `username` - Имя пользователя
- `domain` - Домен пользователя

### Параметры отображения
- `desktopwidth` - Ширина экрана
- `desktopheight` - Высота экрана
- `session bpp` - Битность цвета
- `compression` - Сжатие данных
- `screen mode id` - Режим отображения (окно/полный экран)

### Параметры безопасности
- `authentication level` - Уровень аутентификации
- `enablecredsspssupp` - Использование CredSSP
- `encrypt` - Уровень шифрования

### Параметры производительности
- `connection type` - Тип подключения
- `bandwidthauto` - Автоопределение пропускной способности
- `allow desktop composition` - Использование Aero эффектов
- `networkautodetect` - Автоопределение качества сети

### Перенаправления устройств
- `redirectdrives` - Перенаправление дисков
- `redirectprinters` - Перенаправление принтеров
- `redirectcomports` - Перенаправление портов COM
- `redirectclipboard` - Перенаправление буфера обмена
- `audiomode` - Режим звука
- `audiocapturemode` - Захват микрофона

## RDP файл формат

RDP файлы - это текстовые файлы с параметрами в формате:
```
параметр:тип:значение
```

Где тип может быть:
- `i` - целое число (integer)
- `s` - строка (string)
- `b` - бинарные данные (binary)

Пример:
```
full address:s:192.168.1.10
username:s:administrator
desktopwidth:i:1920
desktopheight:i:1080
```

## Часто задаваемые вопросы

### Как изменить пароль в сохранённом RDP файле?
Рекомендуется использовать опцию "Запроc пароля при подключении" для безопасности. При открытии RDP файла система запросит пароль перед подключением.

### Какое выбрать разрешение экрана?
- Для LAN подключений: используйте максимально возможное разрешение
- Для интернета/мобильных сетей: выбирайте меньшие разрешения (1024x768 или 1366x768) для снижения нагрузки на сеть

### Какой уровень шифрования выбрать?
- Для локальной сети: уровень 1-2
- Для подключений через интернет: уровень 3 (High encryption) или 4 (FIPS)
